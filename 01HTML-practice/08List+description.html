An HTML Ordered List(ol)

  1. Coffee
  2. Tea
  3. Milk

Order List types

  1. You can set type for i to enter Roman Numbers
  2. You can set type for I to enter Roman Numbers(UpperCase)
  3. You can set type for a to enter letters
  4. You can set type for A to enter letters(UpperCase)
  5. You can set type for 1 to enter numbers (default)
  6. You can set start for n to enter numbers started with n
